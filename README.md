# Basic mall project

## 팀 프로젝트
> 아이돌 관련상품을 판매하는 모의 쇼핑몰로, 여러 굿즈 판매사이트를 참고하여 작성

### 개발 환경
- jsp, java 1.8, MySQL
- 개발 기간: 1개월
- 개발 인원: 5명
- 담당: 상품/상품 등록

### 주요 기능
  - 상품 구매/장바구니(실제 결제되지 않습니다)
  - 회원가입/정보수정
  - 이벤트/고객센터 게시판
  - 관리자 상품 및 게시물 등록/수정

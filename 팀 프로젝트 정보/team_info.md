# 팀 프로젝트 

## 개요
> 아이돌 관련상품을 판매하는 모의 쇼핑몰로, 여러 굿즈 판매사이트를 참고하여 작성

## 개발 환경
- jsp, java 1.8, MySQL
- 개발 기간: 1개월
- 개발 인원: 5명
- 담당: 상품/상품 등록
